]]> O problema de cache em dispositivos de armazenamento USB é algo bastante comum e que pode ser frustrante para os usuários. Quando copiamos arquivos grandes para um pendrive ou HD plugado na USB, a cópia é feita rapidamente, mas ao tentarmos remover com segurança o dispositivo, é preciso esperar por um longo período.

Isso ocorre porque a cópia dos arquivos não diferencia a cópia em cache da cópia real para o outro dispositivo. O tamanho da cache varia conforme a quantidade de memória RAM livre, o que é bom para acelerar o sistema, mas pode ser inconveniente para dispositivos USB.

Ao longo dos anos, houve muitos relatos de usuários de várias distribuições Linux que não gostam desse comportamento. No entanto, reduzir a configuração de “dirty pages” para diminuir a cache pode também reduzir a velocidade do sistema como um todo, inclusive do armazenamento interno.

O BigLinux está inovando mais uma vez ao oferecer a solução usb-dirty-pages-udev, que detecta automaticamente dispositivos de armazenamento plugados na USB e estabelece a cache especificamente para eles em 16 MB, enquanto o restante do sistema continua com o sistema de cache padrão. Isso resolve o problema de espera prolongada para remover com segurança os dispositivos USB, sem afetar o desempenho geral do sistema.

Se o BigLinux for instalado em um dispositivo USB, esse dispositivo não terá a redução de cache para 16 MB, mantendo o bom desempenho do sistema.

O post BigLinux 20.04 RC1 lançado com muitas novidades apareceu primeiro em Site Oficial Biglinux.

]]> Mais uma etapa para o BigLinux 20.04 final está concluída, ocorreram vários atrasos, quem é do nosso grupo no Telegram sabe como está sendo trabalhoso trazer uma grande quantidade de novidades para essa versão, que teve ainda mais atrasos gerados pela Covid.

Antes de seguir para um detalhamento maior do sistema, deixo os agradecimentos ao pessoal do grupo do Telegram e do Fórum, principalmente para o Elton Fabricio, que tem tido grande dedicação ao projeto BigLinux, com destaque para as melhorias no BigBashView, o aplicativo de gerenciamento de WebApps , o aplicativo de restaurar programas e o BigLinux Tweaks. Também aos tradutores LordDan e BigQuadra.

Para você que está lendo, fica o convite para participar do nosso grupo (Telegram / Fórum), traga melhorias ao sistema, scripts de automação, interfaces novas, conheça o potencial do BigBashView. Saiba que nosso foco não é ter milhões de usuários ou ficar rico, mas exercitar nosso potencial de desenvolvimento e utilizar o BigLinux como uma forma facilitada e integrada de ver tudo funcionando, se você é um desenvolvedor, seja iniciante ou avançado, compartilhe conosco seu conhecimento e código.

Agora irei listar alguns diferenciais que não estão apenas nesse beta, mas que foram aperfeiçoados também nos últimos meses.

BTRFS OTIMIZADO

Uso de partição Btrfs por padrão, já estamos indo para 2 anos utilizando esse formato, com isso foi possível otimizar as configurações para extrair os melhores recursos e desempenho, junto com um sistema de preload que fizemos com base no vmtouch, o boot do sistema nunca foi tão rápido, além da abertura dos principais programas estar mais rápida.

O uso do Btrfs também reduz o uso de armazenamento, utilizando a compactação Zstd o sistema ocupa 2,9 GB após instalado, enquanto com o tradicional Ext4 é necessário 7 GB.

Ainda temos os snapshots diários, com isso, por padrão, você pode optar voltar o sistema para como estava em qualquer um dos 5 dias anteriores.

BTRFS + DISABLE-FSYNC

Nas últimas semanas fizemos diversos testes utilizando um recurso para desativar o fsync dos programas, com isso o ganho de desempenho do sistema foi muito grande, em contrapartida, caso falte energia e o sistema seja desligado de forma forçada, os arquivos que foram salvos nos últimos segundos irão retornar em sua versão anterior ao salvamento, portanto, é um recurso que compensa ser utilizado na maioria das situações, para mais informações clique aqui.

REDUÇÃO DE RUÍDO DO MICROFONE

Principalmente após a pandemia, o uso de programas de conferência demandam cada vez mais o uso de microfones nos computadores, porém, o microfone de quase todos os computadores possui uma grande quantidade de ruídos, e a maioria das dicas para reduzir esses ruídos não funcionam corretamente. A partir do BigLinux 20.04 beta 31, temos uma interface que configura com poucos cliques um plugin de áudio que realmente reduz muito os ruídos do microfone ou ruídos do ambiente, por exemplo, gerado por aparelhos de ar condicionado. (Link para o plugin utilizado https://github.com/werman/noise-suppression-for-voice )

Se você faz lives, apresentações online, gravações em geral que utilizam microfone, conheça essa versão do BigLinux e veja a diferença que esse plugin faz.

WEBAPPS

São 36 WebApps já configurados por padrão, mas categorizados a parte para não poluir o sistema, além de uma interface simples e prática para que você desative os WebApps, além de poder transformar facilmente qualquer site em um WebApp utilizando nossa interface em BigBashView.

Tudo isso em em um pacote de apenas 2MB.

DESKTOP KDE EM 3 ESTILOS

Utilizamos apenas o Desktop KDE Plasma, porém, desde o modo Live são oferecidas três opções de configuração que fazem o sistema se comportar como desktops diferentes. A qualquer momento no live ou após instalado, na tela de login você pode alternar entre essas três configurações diferentes utilizando o KDE. Se você utiliza notebook não esqueça de conferir o desktop alternativo, que irá aproveitar ao máximo o espaço da tela.

4 OPÇÕES DE TEMA E 3 DE DESEMPENHO

Não basta as opções de desktop, já no boot do modo Live, e também a qualquer momento no sistema instalado, estão disponíveis 4 opções de temas, que com um clique já ajusta todo o sistema, inclusive mantendo harmonia entre os programas QT e GTK.

Talvez você tenha um computador já um pouco antigo e pense que o BigLinux por utilizar KDE não é uma boa opção, porém, nós disponibilizamos configurações específicas para equipamentos de menor capacidade. Temos usuários utilizando em notebook com processador Atom e 2 GB de RAM, que deixaram de utilizar distribuições com XFCE, por ter encontrado melhor desempenho com o KDE no BigLinux.

MAIOR FACILIDADE NA EXECUÇÃO DE PROGRAMAS

Compatibilidade facilitada para programas como Pje Office, Pje Calc, Shodo, IRPF e plugins para bancos.

Facilitações para arquivos .appimage e .jar, então se você clicar duas vezes sobre um arquivo nesses formatos, ele será executado, assim como seria esperado naturalmente por qualquer usuário, sem necessidade de abrir o terminal para executá-los.

RESTAURAR A CONFIGURAÇÃO DOS PROGRAMAS

Interface para recuperar a configuração de programas, bagunçou o navegador, o LibreOfice ou mesmo todo o desktop? Abra nossa interface de recuperação e com poucos cliques estará tudo como novo.

FACILIDADES PARA DESENVOLVEDORES

Facilidades para quem precisa utilizar os servidores Apache e MariaDB, além de diversos editores de código.

INSTALAÇÃO DE KERNEL SIMPLIFICADA

O seu computador precisa de alguma versão específica de kernel? Conheça nossa interface de instalação de kernel e veja dezenas de opções ao alcance de poucos cliques.

OCR PARA ARQUIVOS PDF

Você tem um arquivo em PDF que foi digitalizado e quer fazer uma busca dentro do arquivo ou copiar partes do texto? Utilize nossa interface de OCR, basta clicar com o botão direito do mouse sobre o arquivo PDF, ir em abrir com e selecionar a opção de OCR.

MELHOR RENDERIZAÇÃO DE FONTES

Quantas horas passamos todo dia lendo no computador? Ter fontes de qualidade é essencial e uma das características do BigLinux a mais de 10 anos, sempre utilizando a melhor configuração possível com os recursos disponíveis, nesse beta nossa configuração de fontes está pronta para mais um lançamento, deixando não só o sistema mais agradável, mas todos os sites, já que a renderização dentro dos navegadores também foi aperfeiçoada.

LibreOffice com a extensão LanguageTool, que torna a correção de textos muito mais avançada, inclusive corrigindo erros de concordância. Tenha textos revisados com muito mais qualidade utilizando nossas configurações.

MELHORIAS NA INSTALAÇÃO E ATUALIZAÇÃO DE .DEB

Uma interface muito prática e simples feita para instalar arquivos .deb armazenados localmente, além da nossa tradicional configuração de auto restauração do Apt e Dpkg. Essa configuração está sendo aperfeiçoada a anos e reduz muito a quantidade de problemas durante a instalação de pacotes .deb ou atualizações do sistema.

VERIFICAÇÕES AUTOMÁTICAS PARA RESTABELECER O DESKTOP

Configurações gerais para pequenos erros, por exemplo, se o desktop plasmashell ou o gerenciador de tarefas kwin fecharem, automaticamente o sistema irá executá-los novamente.

OS MELHORES PROGRAMAS, INDEPENDENTE DO DESKTOP

Utilizamos os melhores programas no sistema, por exemplo, a calculadora e parte do gerenciador de redes são do Gnome, juntamos os melhores programas disponíveis para Linux, não nos limitamos a utilizar exclusivamente programas de apenas um desktop.

MAIOR INTEGRAÇÃO ENTRE PROGRAMAS QT E GTK

Sincronização entre diálogos GTK e QT, talvez você já tenha percebido que se adicionar algo aos favoritos no Dolphin, não irá aparecer nos favoritos ao utilizar o Gimp ou qualquer outro programa em GTK, no BigLinux isso é sincronizado.

O CLIQUE DIREITO DO MOUSE, UM PROBLEMA QUE ME INCOMODAVA

O clique do botão direito do mouse, por padrão, no Linux ao clicarmos com o botão direito do mouse, abre o menu com opções, e ao soltar o botão, se estiver em cima de uma das opções do menu, essa opção é executada. Isso gera muitos cliques errados, principalmente para quem tem algum tipo de situação que faça as mãos tremerem, então ao soltar o botão do mouse não é mais executada nenhuma função.

MELHORIAS PARA QUEM UTILIZA MAIS DE UM MONITOR

Melhor suporte a multi monitor, nas últimas semanas fizemos configurações para deixar mais confortável o uso de mais de um monitor, por exemplo, para abrir um programa e ele surgir no mesmo monitor.

INTERFACES E NOMES AMIGÁVEIS

Simplificações da interface dos programas, boa parte dos programas instalados possuem configurações que simplificam o seu uso, leitor de PDF, visualizador de imagens, gerenciador de arquivos e vários outros, ao longo dos anos ao perceber as dificuldades que os usuários relatam vamos personalizando o sistema para deixar tudo cada vez mais simples.

Simplificações dos nomes dos programas, você que está lendo esse texto, deve ter interesse em tecnologia e não teria dificuldade de aprender que Clementine é aonde reproduz áudio, Smplayer vídeo, Gwenview fotos, mas se você instalar esse sistema para uma pessoa que não tem afinidade com tecnologia, a adaptação a esses nomes tende a ser difícil, então todos os programas que já vem instalados no BigLinux passam por uma simplificação, similar ao que é feito na maioria dos Smartphones, então temos, Músicas, Vídeos e Visualizador de Imagens.

Um sistema completo, mas limpo, diversos programas avançados, por exemplo, Gparted e Grub-customizer estão ocultos no menu, se você utilizar a busca do menu eles serão exibidos, ou então utilizando nossa Central de Controle. Com isso facilitamos o uso de recursos avançados, sem deixar o sistema poluído.

UMA FORMA FÁCIL DE ADICIONAR SCRIPTS A UMA INTERFACE GRÁFICA

Muitas soluções para Linux estão disponíveis em forma de scripts, que não são amigáveis para o usuário baixar e executar, porém, criar interfaces amigáveis para esses scripts é um trabalho demorado e tedioso, encontramos um método para resolver esse problema, um programa que basta inserir o script na pasta correta, preencher o nome e a descrição, que assim irá aparecer na interface a opção de executar o script, para uma melhor integração o terminal é exibido de forma embutida no programa. Então essa mensagem é também um convite para você vir fazer parte do BigLinux e trazer seus scripts de automação para nossa interface.

MENOR CONSUMO DE ENERGIA

Consumo de energia, nosso foco em melhorar o desempenho do sistema, que passa por um grande número de otimizações, reflete também no gasto de energia, já que necessita de menos ciclos de processamento e menos gravações em disco, porém, também utilizamos o TLP para fazer otimizações específicas para o consumo de energia.

SUPORTE A PLACA DE VÍDEO SiS

O nosso amigo Maxsandro encontrou a solução para utilizar o BigLinux com esse tipo de placa de vídeo e incluímos na interface BigLinux Tweaks, então basta iniciar o sistema com a opção nomodeset que está em opções de compatibilidade na tela de boot, a resolução estará baixa e o sistema lento, mas após instalar no HD utilize a interface BigLinux Tweaks e clique em instalar o driver SiS, lembre que é preciso estar conectado à internet para baixar o driver.

CONHEÇA E PARTICIPE

Agora que você já sabe um pouco sobre o BigLinux, faça o download, experimente, traga melhorias e vamos em frente nessa busca pelo sistema perfeito que já dura 16 anos…
